AI Quant Letter

## What You Get

Every Monday you receive a data-driven digest covering:

- **Market Season Analysis** — Halving cycle position and monthly seasonal bias
- **Dual Momentum Rankings** — Antonacci's dual momentum adapted for Korean crypto markets
- **Multi-Factor Rankings** — Composite score (momentum + quality + volatility + performance)
- **Weekly Strategy Summary** — Clear position guidance: BUY / HOLD / CASH
- **How It Works** — Transparent methodology with open-source code references

## 6 Quantitative Strategies Implemented

| Strategy | Framework | Key Signal |
|----------|-----------|------------|
| Dual Momentum | Gary Antonacci | Absolute + relative momentum score |
| Volatility Breakout | Larry Williams | Intraday range × k factor |
| Multi-Factor Ranking | Kang Hwan-guk | Composite rank across 4 factors |
| GTAA Asset Allocation | Mebane Faber | Cross-asset momentum allocation |
| Seasonal Filter | Halving cycle timing | Monthly multiplier + cycle phase |
| Risk Management | Position sizing | Stop-loss + volatility-adjusted sizing |

## Open Source

All signals are generated by [`maiupbit`](https://github.com/jini92/M.AI.UPbit),
a Python package published on PyPI under the Apache 2.0 license.

You can reproduce every signal yourself:

    pip install maiupbit
    maiupbit recommend --top 5
    maiupbit quant momentum --top 5
