## 3. 수익 모델 (Tier 구조)

> **현재 상태**: 모든 콘텐츠 무료 공개 중. Stripe 연동 완료 후 유료 tier 활성화 예정.

| Tier | 월 구독료 | 포함 콘텐츠 | 딜레이 |
|------|-----------|-------------|--------|
| **Free** | 무료 | 주간 리포트 (요약판) | 1주 딜레이 |
| **Basic** | $9.9/월 | 주간 리포트 (전체) + 월간 백테스트 리포트 | 즉시 발행 |
| **Pro** | $19.9/월 | Basic 포함 + 매매 시그널 실시간 알림 + Discord 채널 접근 | 즉시 발행 |

### Stripe 연동 현황

| 단계 | 상태 | 비고 |
|------|------|------|
| Wise USD 계좌 신청 | 진행 중 | 발급까지 2~3 영업일 소요 |
| Stripe 계정 생성 | 대기 중 | Wise 계좌 발급 후 진행 |
| Substack → Stripe 연동 | 대기 중 | Stripe "Add Your Bank" 메뉴에서 입력 |
| 유료 tier 활성화 | 대기 중 | Stripe 연동 완료 후 즉시 활성화 |

**Stripe 연동 절차** (Wise 계좌 발급 후):
1. Stripe 계정 생성 (stripe.com)
2. Stripe → Settings → Bank Accounts → Add Your Bank
3. Wise USD 계좌 정보 입력 (Account number, Routing number)
4. Substack → Settings → Payments → Connect Stripe
5. Free / Basic / Pro tier 가격 설정 및 활성화
