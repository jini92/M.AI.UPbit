업비트 주간 퀀트 시그널 — maiupbit 오픈소스 엔진 자동 생성

이번 주 시그널: 현금 유지

이중 모멘텀 상위 5개 코인 전부 마이너스다. 모델이 줄 수 있는 가장 명확한 "매수 금지" 신호다. 로테이션 대상 자체가 없다 — 1위 ETH(-0.057)조차 마이너스 영역이다.

3주 연속 매수 시그널이 나오지 않고 있다.

시장 시즌 컨텍스트

항목 현황
시즌 🟢 강세장 (3월)
할빙 사이클 중기 (할빙 후 695일)
시즌 가중치 1.2×

매크로 시즌 지표는 여전히 강세를 읽고 있다. 하지만 "강세 시즌"이 "지금 다 사라"는 뜻은 아니다. 시즌 가중치는 포지션 사이징을 조정할 뿐, 모멘텀 시그널을 무시하지 않는다.

매크로는 강세인데 단기 모멘텀은 전부 약세 — 이런 긴장 상태에서 시스템적 규칙이 제 역할을 한다. 재량 트레이더는 강세장이라는 이유로 조심성을 무시하기 쉽다. 모델은 그렇지 않다.

이중 모멘텀 TOP 5

순위 코인 점수
🥇 ETH -0.057
🥈 BTC -0.061
🥉 LINK -0.098

점수 읽는 법: 이중 모멘텀은 절대 모멘텀(이 자산이 상승 추세인가?)과 상대 모멘텀(어떤 자산이 가장 강한가?)을 결합한다. 5개 전부 마이너스이므로 절대 모멘텀 기준을 통과하는 자산이 없다. 순위는 덜 나쁜 순서일 뿐이다.

ETH와 BTC가 제로 근처에 밀집해 있다(-0.057 vs -0.061). 다음 주 둘 중 하나가 0 위로 올라오면 3주 만의 첫 로테이션 시그널이 된다.

1 ETH 0.554
2 AVAX 0.527

멀티팩터 모델(추세, 변동성, 거래량, 온체인 지표 조합)은 더 낙관적이다. ETH와 AVAX가 0.5 이상으로, 순수 모멘텀이 암시하는 것만큼 약하지는 않다는 뜻이다.

모멘텀(전부 마이너스)과 멀티팩터(대체로 플러스)의 괴리는 횡보 구간에서 자주 나타난다 — 하락은 멈췄지만 새 상승을 시작하지 않은 상태다.

지난주 대비 변화

  • ETH 두 모델 모두 1위 유지. 서로 다른 시그널 유형에서 같은 자산이 선두라면, 다음 로테이션 시 가장 유력한 후보다.
  • SOL 두 모델 모두 최하위. 멀티팩터 점수(0.255)와 1위(0.554) 간 격차가 TOP 5 중 가장 크다.
  • AVAX 멀티팩터 2위로 상승 — 모멘텀은 부진해도 펀더멘털 개선 시사.

시스템의 강점

3주 연속 "현금 유지"는 지루하다. 그게 핵심이다.

maiupbit 퀀트 엔진은 화제가 되는 뉴스가 없어도 매주 돌아간다. 뉴스를 쫓지 않는다. 횡보 구간에서 조급해하지 않는다. 수학이 확인될 때까지 기다린다.

직접 퀀트 시그널을 만들고 싶다면, 엔진 전체가 오픈소스다:

  • PyPI: pip install maiupbit
  • GitHub: github.com/jini92/M.AI.UPbit
  • 스택: Python + LSTM + GPT-4o 분석 + UPbit API
