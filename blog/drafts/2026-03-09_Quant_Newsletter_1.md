## 🔧 이 데이터는 어떻게 만들어지나요?

이 레터는 제가 개발한 [maiupbit](https://pypi.org/project/maiupbit/) 패키지로 자동 생성됩니다.

- **강환국 퀀트 전략 6종** 구현 (듀얼 모멘텀, 변동성 돌파, 멀티팩터, GTAA, 시즌 필터, 리스크 관리)
- **업비트 API** 실시간 연동
- **AI 분석**: OpenAI GPT-4o + 로컬 LLM (Ollama)
- **완전 오픈소스**: Apache-2.0 라이선스

`ash
pip install maiupbit
maiupbit analyze KRW-BTC
`
